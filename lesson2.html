Make your pages beautiful with colors, fonts, and layout.

Colors

CSS colors let you change the appearance of text, backgrounds, and other elements to make a webpage more visually appealing and easier to read.

  • You can use simple color names like blue, pink, or gray for quick styling.
  • More precise colors can be created using hex codes (like #ff5733) or RGB values (rgb(255, 87, 51)).
  • Colors can be applied to different properties such as color (text color) and background-color (background).

Fonts

CSS fonts let you control how text looks on a webpage, including its style, size, and overall readability.

  • You can choose a font using font-family, such as Arial, Times New Roman, or more modern web fonts.
  • Use font-size to control how big or small the text appears.
  • Properties like font-weight (boldness) and font-style (e.g., italic) help emphasize important content.

Padding

CSS padding controls the space inside an element, between its content and its border, helping make layouts look less cramped and more readable.

  • You can apply padding to all sides at once using padding, or control each side individually with padding-top, padding-right, etc.
  • Increasing padding creates more space around the content, making elements easier to see and interact with.
  • You can use values like pixels (px) or percentages depending on your layout needs.
